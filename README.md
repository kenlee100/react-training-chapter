# React 直播班 每週任務

#### 啟動專案
進入各週目錄下安裝 npm 套件，然後啟動專案

```
cd week2
npm i
npm run dev
```

#### 部署
進入各週目錄下部署專案
```
cd week2
npm run deploy
```
